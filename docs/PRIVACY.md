# Privacy Policy - GitHub Guide Tool

Last updated: February 2025

## Overview

GitHub Guide Tool is a Chrome extension that analyzes GitHub repositories and generates AI-powered documentation. We are committed to protecting your privacy.

## Data Collection

### What We Collect
- **GitHub repository data**: Public repository information (file structure, code snippets) accessed through GitHub API
- **User settings**: API keys and preferences stored locally in Chrome storage

### What We Don't Collect
- Personal identification information
- Browsing history outside of GitHub
- Analytics or tracking data

## Data Storage

- All data is stored locally on your device using Chrome's storage API
- API keys are stored in Chrome's local storage and never transmitted to our servers
- Analysis results are cached locally for 24 hours

## Third-Party Services

This extension connects to the following third-party AI services based on your configuration:
- Anthropic (Claude API)
- OpenAI API
- SiliconFlow API
- BigModel (智谱 AI) API

Your repository data is sent to your selected AI provider for analysis. Please review their respective privacy policies:
- [Anthropic Privacy Policy](https://www.anthropic.com/privacy)
- [OpenAI Privacy Policy](https://openai.com/privacy)
- [SiliconFlow Privacy Policy](https://siliconflow.cn/privacy)
- [BigModel Privacy Policy](https://open.bigmodel.cn/privacy)

## Data Security

- API keys are stored locally and transmitted only to their respective services
- All API communications use HTTPS encryption
- No data is stored on external servers controlled by us

## Your Rights

You can:
- Clear all stored data by removing the extension
- View and modify your settings at any time
- Choose which AI service to use

## Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected in the "Last updated" date.

## Contact

For questions about this privacy policy, please open an issue on our [GitHub repository](https://github.com/alexanderkinging/github-guide-tool).
